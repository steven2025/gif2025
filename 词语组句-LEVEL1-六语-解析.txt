第 1 题
人 哪 是 国 你
正确答案：你是哪国人

答案解析（中文）
正确答案：你是哪国人。
“你”为主语；系词“是”；“哪”作定语修饰“国”，与“人”构成疑问名词短语“哪国人”。结构：主语 + 系词 + 表语（疑问名词短语）。
技巧/tips:
找主语：你
立系词：用“是”连接主表
设疑问：把“哪”放在“国”前，构成“哪国（人）”

答案解析（英文）
The correct answer is ：你是哪国人
EN: Which country are you from? / What nationality are you?
Subject 你 (you) + copula 是 (are) + interrogative nominal 哪国人 (“person of which country”). Word order: Subject + Copula + Predicate (interrogative NP).
Tips: Identify 你 → choose copula 是 → build 哪 + 国 + 人.

答案解析（法语）
The correct answer is ：你是哪国人
FR：Tu es de quel pays ? / De quelle nationalité es-tu ?
« 你 » = sujet ; « 是 » = copule ; « 哪 + 国 + 人 » forme le GN interrogatif 哪国人. Schéma : Sujet + Copule + GN interrogatif.
Tips: Sujet 你 → copule 是 → placer 哪 devant 国 pour former 哪国人。

答案解析（泰语）
The correct answer is ：你是哪国人
TH：คุณเป็นคนประเทศไหน / คุณสัญชาติอะไร
ประธาน คุณ + กริยาเชื่อม เป็น + นามเชิงคำถาม คนประเทศไหน.
Tips: ระบุประธาน คุณ → ใช้ เป็น → สร้าง ไหน + ประเทศ + คน.

答案解析（俄语）
The correct answer is ：你是哪国人
RU: Ты из какой страны? / Какой ты национальности?
Подлежащее ты + связка есть/являешься (是) + именная часть-вопрос «какой страны (человек)».
Tips: Подлежащее ты → связка 是 → построить «какой + страна + человек».

答案解析（老挝语）
The correct answer is ：你是哪国人
LO: ເຈົ້າເປັນຄົນປະເທດໃດ?
«ເຈົ້າ» = ປະທານ; «ເປັນ» = ກິລິຍາເຊື່ອມ; «ຄົນ + ປະເທດໃດ» = ນາມຖາມ.
Tips: ກໍານົດປະທານ ໐ເຈົ້າ → ໃຊ້ ໐ເປັນ → ປະກອບ ໐ໃດ + ປະເທດ + ຄົນ.
