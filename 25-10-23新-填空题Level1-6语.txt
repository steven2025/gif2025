第1题
人 英国 国家

句子1. 你来自哪个(   )？
正确答案: 国家

句子2. 街上很多(    )。
正确答案: 人

句子3. 你去过(    )吗？
正确答案: 英国

答案解析（中文）

[句1] 完成句：你来自哪个**国家**？ 理由： “哪个+类别名词”问类型 → 选“国家”。

[句2] 完成句：街上很多**人**。 理由： “很多”修饰人群数量 → 选“人”。

[句3] 完成句：你去过**英国**吗？ 理由： “去过+国家/地名”固定搭配 → 选“英国”。

答案解析（英文）

[S1] CN：你来自哪个**国家**？ EN：Which **country** are you from?
Why: “Which + category” asks for type → choose country.

[S2] CN：街上很多**人**。 EN：There are many **people** on the street.
Why: “many” modifies a people group → people.

[S3] CN：你去过**英国**吗？ EN：Have you been to the **UK**?
Why: “have been to + country name” → UK.

技巧 / Tips：
Type→**country**; crowd→**people**; been to→**UK**.

答案解析（法语）

[S1] CN：你来自哪个**国家**？ FR：Tu viens de quel **pays** ?
Pourquoi : « quel + catégorie » interroge le type → **pays**.

[S2] CN：街上很多**人**。 FR：Il y a beaucoup de **gens** dans la rue.
Pourquoi : « beaucoup de » qualifie un groupe de personnes → **gens**.

[S3] CN：你去过**英国**吗？ FR：Tu es déjà allé(e) au **Royaume‑Uni** ?
Pourquoi : « être allé + nom de pays » → **Royaume‑Uni**.

技巧 / Tips：
Type→**pays**；foule→**gens**；déjà allé→**Royaume‑Uni**.

答案解析（泰语）

[1] CN：你来自哪个**国家**？ TH：คุณมาจาก**ประเทศ**ไหน?
เหตุผล: “ไหน + ประเภท” ถามชนิด → ประเทศ

[1] CN：街上很多**人**。 TH：บนถนนมี**ผู้คน**มาก
เหตุผล: คำบอกปริมาณขยายกลุ่มคน → ผู้คน

[1] CN：你去过**英国**吗？ TH：คุณเคยไป**อังกฤษ**ไหม?
เหตุผล: “เคยไป + ชื่อประเทศ” → อังกฤษ

技巧 / Tips：
ประเภท→**ประเทศ**; คนเยอะ→**ผู้คน**; เคยไป→**ประเทศ**.

答案解析（俄语）

[П1] CN：你来自哪个**国家**？ RU: Из какой **страна** ты?
Причина: «который?» + категория → страна

[П2] CN：街上很多**人**。 RU: На улице много **люди**.
Причина: Количество людей → люди

[П3] CN：你去过**英国**吗？ RU: Ты бывал в **Великобритания**?
Причина: «бывал в + страна» → Великобритания

技巧 / Tips：
тип→**страна**; много людей→**люди**; «бывал в»→**страна**.

答案解析（老挝语）

[1] CN：你来自哪个**国家**？ LO：ເຈົ້າມາຈາກ**ປະເທດ**ໃດ?
ເຫດຜົນ: “ໃສ + ປະເພດ” ຖາມປະເພດ → ປະເທດ

[2] CN：街上很多**人**。 LO：ບນຖະໜົນມີ**ຄົນ**ຫຼາຍ
ເຫດຜົນ: ຈໍານວນຫຼາຍສໍາລັບກຸ່ມຄົນ → ຄົນ

[3] CN：你去过**英国**吗？ LO：ເຈົ້າເຄີຍໄປ**ອັງກິດ**ບໍ?
ເຫດຜົນ: “ເຄີຍໄປ + ຊື່ປະເທດ” → ອັງກິດ

技巧 / Tips：
ປະເພດ→**ປະເທດ**; ຄົນຫຼາຍ→**ຄົນ**; ເຄີຍໄປ→**ປະເທດ**.

第2题
请 请问 叫

句子1. (   )，学校在哪？
正确答案: 请问

句子2. (    )坐。这里有椅子。
正确答案: 请

句子3. 你(    )什么名字？
正确答案: 叫

答案解析（中文）

[句1] 完成句：**请问**，学校在哪？ 理由： 句首礼貌用语，引出提问 → 选“请问”。

[句2] 完成句：**请**坐。这里有椅子。 理由： “请+动词”表示礼貌邀请/请求 → 选“请”。

[句3] 完成句：你**叫**什么名字？ 理由： “叫+名字”用于询问姓名 → 选“叫”。

答案解析（英文）

[S1] CN：**请问**，学校在哪？ EN：**may i ask**, where is the school?
Why: Polite opener before a question → “may I ask”.

[S2] CN：**请**坐。这里有椅子。 EN：**please** have a seat. There is a chair here.
Why: “please + verb” polite request → please.

[S3] CN：你**叫**什么名字？ EN：What are you **to be called**?
Why: “be called + name” to ask someone’s name → called.

技巧 / Tips：
Polite opener→**may I ask**; request→**please**; ask name→**to be called**.

答案解析（法语）

[S1] CN：**请问**，学校在哪？ FR：**excusez‑moi**, où est l’école ?
Pourquoi : formule polie d’ouverture avant une question → « excusez‑moi / puis‑je demander ».

[S2] CN：**请**坐。这里有椅子。 FR：**s’il vous plaît**, asseyez‑vous. Il y a une chaise ici.
Pourquoi : « s’il vous plaît + verbe » = demande polie → **s’il vous plaît**.

[S3] CN：你**叫**什么名字？ FR：Comment **t’appelles‑tu** ?
Pourquoi : « s’appeler + nom » pour demander le nom → **s’appeler**.

技巧 / Tips：
Ouverture polie→**excusez‑moi**；demande→**s’il vous plaît**；demander le nom→**s’appeler**.

答案解析（泰语）

[1] CN：**请问**，学校在哪？ TH：**ขอถามหน่อย** โรงเรียนอยู่ที่ไหน?
เหตุผล: วลีสุภาพขึ้นต้นคำถาม → ขอถามหน่อย

[1] CN：**请**坐。这里有椅子。 TH：**เชิญ**นั่ง ตรงนี้มีเก้าอี้
เหตุผล: “เชิญ/กรุณา + กริยา” → เชิญ

[1] CN：你**叫**什么名字？ TH：คุณ**ชื่อ**อะไร?
เหตุผล: ถามชื่อ → ชื่อ

技巧 / Tips：
ขึ้นต้นสุภาพ→**ขอถามหน่อย**; ขอร้อง→**เชิญ**; ถามชื่อ→**ชื่อ**.

答案解析（俄语）

[П1] CN：**请问**，学校在哪？ RU: **скажите, пожалуйста**, где находится школа?
Причина: Вежливое вступление к вопросу → «скажите, пожалуйста».

[П2] CN：**请**坐。这里有椅子。 RU: **пожалуйста**, садитесь. Здесь есть стул.
Причина: «пожалуйста + действие» вежливая просьба.

[П3] CN：你**叫**什么名字？ RU: Как тебя **зовут**?
Причина: «зовут + имя» спрашивает имя.

技巧 / Tips：
Вежливое начало→**скажите, пожалуйста**; просьба→**пожалуйста**; имя→**зовут**.

答案解析（老挝语）

[1] CN：**请问**，学校在哪？ LO：**ຂໍຖາມແນ່**, ໂຮງຮຽນຢູ່ໃສ?
ເຫດຜົນ: ຄໍານໍາຖາມຢ່າງສຸພາບ → ຂໍຖາມແນ່

[2] CN：**请**坐。这里有椅子。 LO：**ກະລຸນາ/ເຊີນ**ນັ່ງ ທີ່ນີ້ມີເກົ້າອີ້
ເຫດຜົນ: “ກະລຸນາ/ເຊີນ + ກິລິຍາ” → ກະລຸນາ/ເຊີນ

[3] CN：你**叫**什么名字？ LO：ເຈົ້າ**ຊື່**ຫຍັງ?
ເຫດຜົນ: ຖາມຊື່ → ຊື່

技巧 / Tips：
ນໍາຄໍາຖາມ→**ຂໍຖາມແນ່**; ຂໍຮ້ອງ→**ກະລຸນາ/ເຊີນ**; ຊື່→**ຊື່**.

第3题
吗 什么 哪 

句子1. 你喜欢吃(    )？
正确答案: 什么

句子2. (    )一个好？
正确答案: 哪

句子3. 你吃饱了(    )？
正确答案: 吗

答案解析（中文）

[句1] 完成句：你喜欢吃什么？ 理由：问内容 → “什么”。

[句2] 完成句：哪一个好？ 理由：问选择 → “哪”。

[句3] 完成句：你吃饱了吗？ 理由：是非问句句末助词 → “吗”。

技巧 / Tips：
内容→什么；选择→哪；是非→吗。

答案解析（英文）

[S1] CN：你喜欢吃什么？ EN：What do you like to eat?
Why: content question → what.

[S2] CN：哪一个好？ EN：Which one is better?
Why: choice question → which.

[S3] CN：你吃饱了吗？ EN：Are you full (yes–no question)?
Why: sentence-final yes–no particle → ma.

技巧 / Tips：
what / which / yes–no (ma).

答案解析（法语）

[S1] CN：你喜欢吃什么？ FR：Qu’est‑ce que tu aimes manger ?
Pourquoi : question de contenu → **quoi**.

[S2] CN：哪一个好？ FR：Lequel est meilleur ?
Pourquoi : question de choix → **lequel**.

[S3] CN：你吃饱了吗？ FR：Tu es **rassasié(e)** ?
Pourquoi : particule interrogative finale de type oui/non en chinois « **吗** » → question fermée.

技巧 / Tips：
**quoi** / **lequel** / oui‑non (« **吗** »).

答案解析（泰语）

[1] CN：你喜欢吃什么？ TH：คุณชอบกินอะไร
เหตุผล: ถามเนื้อหา → อะไร

[2] CN：哪一个好？ TH：อันไหนดีกว่า?
เหตุผล: ถามตัวเลือก → อันไหน

[3] CN：你吃饱了吗？ TH：คุณอิ่มแล้วไหม?
เหตุผล: คำช่วยถามใช่/ไม่ → ไหม

技巧 / Tips：
อะไร / อันไหน / ไหม.

答案解析（俄语）

[П1] CN：你喜欢吃什么？ RU：Что ты любишь есть?
Причина: вопрос о содержании → что

[П2] CN：哪一个好？ RU：Который лучше?
Причина: выбор → который

[П3] CN：你吃饱了吗？ RU：Ты сыт? (вопрос «да/нет»)
Причина: вопросительная частица да/нет в китайском → ма

技巧 / Tips：
что / который / вопрос да–нет.

答案解析（老挝语）

[1] CN：你喜欢吃什么？ LO：ເຈົ້າມັກກິນຫຍັງ
ເຫດຜົນ: ຖາມເນື້ອໃນ → ຫຍັງ

[2] CN：哪一个好？ LO：ອັນໃດດີ?
ເຫດຜົນ: ຖາມເລືອກ → ອັນໃດ

[3] CN：你吃饱了吗？ LO：ເຈົ້າອິ່ມແລ້ວບໍ?
ເຫດຜົນ: ຄໍາຊ່ວຍຖາມ “ແມ່ນ/ບໍ?” → ບໍ

技巧 / Tips：
ຫຍັງ / ອັນໃດ / ບໍ?

第4题
我 他 她

句子1. (    )是男生。
正确答案: 他

句子2. (    )很高兴认识你。
正确答案: 我

句子3. （  ）住在女生宿舍。
正确答案: 她

答案解析（中文）

[句1] 完成句：**他**是男生。 理由： “男生”对应男性第三人称 → 选“他”。

[句2] 完成句：**我**很高兴认识你。 理由： 自称用第一人称 → 选“我”。

[句3] 完成句：**她**住在女生宿舍。 理由： “女生宿舍”对应女性主语 → 选“她”。

答案解析（英文）

[S1] CN：**他**是男生。 EN：He is a boy.
Why: “male student” → he.

[S2] CN：**我**很高兴认识你。 EN：I’m glad to meet you.
Why: Self-reference → I.

[S3] CN：**她**住在女生宿舍。 EN：She lives in the girls’ dorm.
Why: Female subject → she.

技巧 / Tips：
**he**→male; **I**→self; **she**→female.

答案解析（法语）

[S1] CN：**他**是男生。 FR：**Il** est un garçon / C’est un garçon.
Pourquoi : sujet masculin → **il**.

[S2] CN：**我**很高兴认识你。 FR：**Je** suis très heureux/heureuse de te connaître.
Pourquoi : auto‑référence (1ʳᵉ personne) → **je**.

[S3] CN：**她**住在女生宿舍。 FR：**Elle** habite au dortoir des filles.
Pourquoi : sujet féminin → **elle**.

技巧 / Tips：
**il**→masculin；**je**→soi；**elle**→féminin.

答案解析（泰语）

[1] CN：**他**是男生。 TH：เขา(ผู้ชาย)เป็นผู้ชาย/นักเรียนชาย
เหตุผล: ลักษณะเพศชาย → เขา(ผู้ชาย)

[1] CN：**我**很高兴认识你。 TH：ฉันดีใจที่ได้รู้จักคุณ
เหตุผล: อ้างตนเอง → ฉัน

[1] CN：**她**住在女生宿舍。 TH：เธออยู่หอพักหญิง
เหตุผล: ประธานเป็นผู้หญิง → เธอ

技巧 / Tips：
**เขา(ผู้ชาย)**→เพศชาย; **ฉัน**→ตนเอง; **เธอ**→เพศหญิง.

答案解析（俄语）

[П1] CN：**他**是男生。 RU: Он — мальчик (студент).
Причина: Пол мужской → он.

[П2] CN：**我**很高兴认识你。 RU: Я рад познакомиться.
Причина: Самоопределение → я.

[П3] CN：**她**住在女生宿舍。 RU: Она живёт в общежитии для девушек.
Причина: Женский род → она.

技巧 / Tips：
**он**→муж.; **я**→сам; **она**→жен.

答案解析（老挝语）

[1] CN：**他**是男生。 LO：ເຂົາ (ຜູ້ຊາຍ) ແມ່ນນັກຮຽນຊາຍ
ເຫດຜົນ: ເພດຊາຍ → ເຂົາ

[2] CN：**我**很高兴认识你。 LO：ຂ້ອຍດີໃຈທີ່ໄດ້ຮູ້ຈັກເຈົ້າ
ເຫດຜົນ: ເວົ້າຖຶງຕົນເອງ → ຂ້ອຍ

[3] CN：**她**住在女生宿舍。 LO：ນາງຢູ່ຫໍພັກຍິງ
ເຫດຜົນ: ເພດຍິງ → ນາງ

技巧 / Tips：
**ເຂົາ**→ຜູ້ຊາຍ; **ຂ້ອຍ**→ຕົນເອງ; **ນາງ**→ຍິງ.

第5题
要 想 有

句子1. 留学生小明(    )吃中国菜。
正确答案: 想

句子2. 你(    )喝咖啡还是牛奶？
正确答案: 要

句子3. 你(    )兄弟姐妹吗？
正确答案: 有

答案解析（中文）

[句1] 完成句：留学生小明**想**吃中国菜。 理由： “想+动词”表达愿望 → 选“想”。

[句2] 完成句：你**要**喝咖啡还是牛奶？ 理由： 提供选择，“要+名词/动词” → 选“要”。

[句3] 完成句：你**有**兄弟姐妹吗？ 理由： 表示拥有/存在 → 选“有”。

答案解析（英文）

[S1] CN：留学生小明**想**吃中国菜。 EN：Xiao Ming the international student wants to eat Chinese food.
Why: Desire/intention before verb → want to.

[S2] CN：你**要**喝咖啡还是牛奶？ EN：Do you want coffee or milk?
Why: Offer/choice uses “want + N” → want.

[S3] CN：你**有**兄弟姐妹吗？ EN：Do you have any siblings?
Why: Possession → have.

技巧 / Tips：
**want to**→desire; **want**→choice/offer; **have**→possession.

答案解析（法语）

[S1] CN：留学生小明**想**吃中国菜。 FR：L’étudiant étranger Xiao Ming **veut** manger de la cuisine chinoise.
Pourquoi : désir/intention avant le verbe → **vouloir**.

[S2] CN：你**要**喝咖啡还是牛奶？ FR：Tu **veux** du café ou du lait ?
Pourquoi : offre/choix avec « vouloir + nom » → **vouloir**.

[S3] CN：你**有**兄弟姐妹吗？ FR：Tu **as** des frères et sœurs ?
Pourquoi : possession → **avoir**.

技巧 / Tips：
**vouloir**→désir/choix；**avoir**→possession.

答案解析（泰语）

[1] CN：留学生小明**想**吃中国菜。 TH：เสี่ยวหมิงนักศึกษาต่างชาติอยากกินอาหารจีน
เหตุผล: ความต้องการก่อนกริยา → อยาก

[1] CN：你**要**喝咖啡还是牛奶？ TH：คุณต้องการดื่มกาแฟหรือ นม?
เหตุผล: เสนอทางเลือก “ต้องการ + N” → ต้องการ

[1] CN：你**有**兄弟姐妹吗？ TH：คุณมีพี่น้องไหม?
เหตุผล: การครอบครอง → มี

技巧 / Tips：
**อยาก**→ความต้องการ; **ต้องการ**→เลือก; **มี**→ครอบครอง.

答案解析（俄语）

[П1] CN：留学生小明**想**吃中国菜。 RU: Иностранец Сяо Мин хочет поесть китайской еды.
Причина: Желание перед глаголом → хотел бы.

[П2] CN：你**要**喝咖啡还是牛奶？ RU: Ты хочешь кофе или молоко?
Причина: Выбор «хочу + сущ.» → хочу.

[П3] CN：你**有**兄弟姐妹吗？ RU: У тебя есть братья или сёстры?
Причина: Наличие → есть.

技巧 / Tips：
**хотел бы**→намерение; **хочу**→выбор; **есть**→наличие.

答案解析（老挝语）
